# WordSprout - 单词萌芽

WordSprout 单词萌芽是一个为儿童设计的英语单词学习的微信小程序产品原型旨在通过有趣的交互和游戏化元素，让儿童轻松愉快地学习英语单词。

使用纯HTML+CSS+JS实现，由Cursor AI编码工具生成，然后通过CloudFlare提供的WorkersAndPages部署发布提供的公网URL,借助Figma的html.to.design插件导入Figma进行二次开发。

项目名称 "WordSprout"（单词萌芽）寓意着知识的种子在孩子心中萌芽和成长：
- **Word**：直指英语学习核心
- **Sprout**：象征少儿如幼苗般在单词灌溉下成长
- **自然联想**：萌芽过程暗合遗忘曲线复习机制
- **品牌延展**：可设计种子→幼苗→大树的成长体系

## 项目概览

这是一个使用HTML、CSS和JavaScript构建的高保真原型，专为微信小程序平台设计，可以直接导入Figma进行进一步设计和开发。

## 运行项目

1. 克隆本仓库到本地
2. 打开项目根目录下的 `index.html` 文件：
   - 直接在浏览器中打开
   - 或使用VS Code的Live Server插件等本地服务器运行

## 预览方式

项目提供了两种预览方式：

1. **移动设备模式**：访问 `src/preview.html`
   - 提供移动设备视图
   - 可通过下拉菜单切换不同页面查看效果

2. **网格视图**：访问 `src/gallery.html`
   - 在网格布局中查看所有页面
   - 适合项目总览和一次性展示所有页面
   - 方便导入Figma进行二次编辑

## 功能模块

| 模块 | 说明 | 实现方式 |
|------|------|---------|
| 基础单词学习 | 通过卡片形式展示单词、图片、发音、例句等 | 分级单词卡(CEFR标准)、预设学习路径(A/B/C三级)、固定例句库、本地JSON词库、音频发音 |
| 语音训练 | 训练儿童正确发音，AI评测发音准确度 | 语音识别、波形动效、评分反馈、情景对话模板、阈值判断评分(60/80/100分段) |
| 记忆游戏 | 通过游戏化方式加深记忆 | 单词配对游戏、听力选择题、拼写挑战、错题本本地存储 |
| 数据看板 | 追踪学习进度和成就 | 学习进度统计、正确率趋势图、可视化图表、勋章系统(铜/银/金三阶) |

## 原型页面

项目包含以下核心页面：

- **首页**：应用主入口，展示学习进度和主要功能
  - 今日学习进度条
  - 签到打卡日历
  - 快捷复习入口
- **单词学习页**：单词卡片学习界面，含发音、例句等
  - 单词卡片展示
  - 发音按钮
  - 例句展示
  - 掌握程度选择
- **难度选择页**：根据年龄和水平选择适合的难度
  - CEFR标准可视化展示(A1-C2)
  - 每个级别配套能力描述
  - 解锁状态管理
- **语音训练页**：朗读单词并获得发音评分
  - 麦克风按钮
  - 波形可视化
  - 评分星星
  - 发音建议
- **游戏中心**：多种记忆游戏选择
  - 单词配对游戏
  - 听力选择题
  - 拼写闯关
  - 单词连连看
- **报告页**：学习数据统计和可视化
  - 连续学习天数
  - 各游戏类型正确率对比
  - 单词掌握趋势图
- **个人中心**：用户信息和成就展示
  - 用户头像/昵称
  - 成就徽章展示
  - 功能入口
- **设置页**：应用设置和偏好配置
- **错题本**：错误单词记录和复习
- **成就系统页**：展示用户获得的所有成就和徽章
- **单词连连看**：词汇与图片匹配的游戏页面
- **对话练习**：情景对话训练页面
- **收藏单词**：用户收藏的单词列表
- **学习计划**：个性化学习计划设置

## 技术栈

- **HTML5**：页面结构和语义化标签
- **Tailwind CSS**：用于快速构建现代界面
- **JavaScript**：交互功能和逻辑
- **Font Awesome**：图标库

## 交互细节

1. **全局交互标准**：
   - 页面切换动画
   - 按钮点击缩放效果
   - 统一的错误处理提示

2. **单词学习流程**：
   - 单词卡片展示→发音按钮→例句展示→掌握程度选择→自动跳转下个单词

3. **游戏交互**：
   - 单词配对：卡片翻转动画+计时器
   - 听力选择：音频播放+选项选择
   - 拼写闯关：虚拟键盘输入+错误提示
   - 单词连连看：词汇与图片配对+连线动画

## 设计说明

1. **色彩方案**：使用绿色为主色调，代表生长和希望，符合"萌芽"的概念
2. **界面设计**：简洁明快，符合儿童审美和使用习惯
3. **交互体验**：操作简单直观，反馈及时明确
4. **可访问性**：考虑了不同年龄段儿童的使用需求
5. **视觉层次**：清晰的视觉层次，重点内容突出

## 重要说明

- 这是一个简易产品原型的纯HTML+CSS展示

- 语音识别和评分功能需要接入专业API
- 数据存储和用户认证需要后端支持
